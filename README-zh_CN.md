# fgit

一个在克隆项目时自动把 GitHub 链接替换为 fastgit 链接的工具。

[English](./README.md)

## 如何使用？

### Linux
1. 克隆本项目
2. 在项目文件夹中执行: `sudo cp ./scripts/fgit.sh /usr/local/bin`
3. 使用 `fgit https://github.com/xxx/xxx.git` 来克隆一个项目

### Windows
1. 克隆本项目
2. 将 fgit/scripts 加入到你的环境变量 (PATH) 中
3. 使用 `fgit https://github.com/xxx/xxx.git` 来克隆一个项目
